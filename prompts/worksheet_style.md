Ruberututor Worksheet Style (House Style)

- Format: Letter size, 2-column for item-dense pages.
- Header: Skill code + micro-example + time box (mm:ss).
- Variants: A (scaffolded), B (standard), C (mixed review).
- Item writing: short stems, minimal distractions, cumulative review every 5th item.
- Answer keys separate; include worked examples for A-variants.
- Typography: Inter (text), STIX Two Math (symbols).
- Do NOT copy Kumon content or trade dress; all content must be original.
